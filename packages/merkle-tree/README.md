# merkle-tree

Merkle tree implementation for the asset database.

This package provides a balanced merkle tree data structure for efficient file comparison and synchronization.

## Setup

Open a terminal and change directory to the merkle-tree project:

```bash
cd photosphere/packages/merkle-tree
```

Install dependencies:

```bash
bun install
```

## Compile

Compile the code:

```bash
bun run compile
```

Compile with live reload:

```bash
bun run compile:watch
```

## Run automated tests

```bash
bun test
```
