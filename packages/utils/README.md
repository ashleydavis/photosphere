# utils

Various utility functions.

## Setup

Open a terminal and change directory to the user-interface project:

```bash
cd photosphere-monorepo/packages/utils
```

Install dependencies:

```bash
pnpm install
```

## Compile

Compile the code:

```bash
pnpm compile
```

Compile with live reload:

```bash
pnpm run compile:watch
```

## Run automated tests

```bash
pnpm test
```
