# api

Platform-agnostic API types and utilities for the Photosphere media file database. Contains shared interfaces, types, and logic that is not specific to any runtime environment.

## Commands

- `bun run compile` - Compile TypeScript
- `bun run compile:watch` - Compile in watch mode
- `bun run test` - Run tests
- `bun run test:watch` - Run tests in watch mode
- `bun run test:coverage` - Run tests with coverage
- `bun run clean` - Remove build artifacts
